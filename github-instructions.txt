How to Push Your TravelScore Project to GitHub
===========================================

1. Create a new GitHub repository:
   - Go to https://github.com/new in your browser
   - Name it 'travelscore' 
   - Add an optional description
   - Choose public or private visibility
   - Do NOT initialize with README, .gitignore, or license

2. Copy the repository URL shown after creation
   It should look like: https://github.com/yourusername/travelscore.git

3. Open a terminal in your project folder and run these commands:
   ```
   git remote set-url origin YOUR_COPIED_URL
   git push -u origin main
   ```
   
   Replace YOUR_COPIED_URL with the URL of your new GitHub repository.

4. When prompted, enter your GitHub credentials

5. Your code will now be uploaded to GitHub

If you prefer to use the GitHub Desktop application:
1. Download and install GitHub Desktop from https://desktop.github.com/
2. Open GitHub Desktop and clone your repository
3. Drag and drop your project files into the repository folder
4. Commit and push your changes
